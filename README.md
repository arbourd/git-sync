# git-sync

`git-sync` updates all local branches from remotes.

## Installation

Install with `brew`.

```console
$ brew tap arbourd/tap
$ brew install git-sync
```

## Usage

Update your branches.

```console
$ git sync
Updated branch main (was 8915328).
Updated branch feature-123 (was 7c24329).
```
